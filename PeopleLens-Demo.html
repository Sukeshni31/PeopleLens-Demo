Sync from Google Sheets
In your Google Sheet → click PeopleLens → Export to PeopleLens → click Copy All → paste below.
One-time setup (2 minutes):
1 Open your Training Google Sheet
2 Click Extensions → Apps Script
3 Delete existing code → paste the PeopleLens_Sync_Script.gs file
4 Make sure line 8 says SHEET_TYPE = 'TRAINING'
5 Click Save → Run → Allow permissions
6 Refresh your sheet → PeopleLens → Export menu appears
7 Repeat for Resource sheet with SHEET_TYPE = 'RESOURCE'
